# Қара Қатын Telegram Bot

`mushka-bot` стиліне ұқсас бір файлдық TypeScript Telegram бот.

## Орнату

```bash
npm install
cp .env.example .env
# BOT_TOKEN мәнін .env ішіне жазыңыз
npm run build
npm start
```

## Командалар

- `/newgame` - осы чатта ойын бөлмесін құру немесе қайта бастау
- `/join` - ойын бөлмесіне қосылу
- `/leave` - ойын басталмай тұрып шығу
- `/startgame` - ойынды бастау, тек ойын иесіне рұқсат

Ойын inline батырмалар арқылы жүреді. Карталар inline панельде стикер болып көрінеді. Ойындар жадыда сақталады, сондықтан бот қайта қосылса белсенді ойындар өшеді.
